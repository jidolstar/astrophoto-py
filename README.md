# astrophoto-py

Python 3.9 환경에서 실행합니다. <br/>
app.py를 실행하면 됩니다. <br/>
원본 이미지는 resources 폴더에 H-alpha, Oiii, Sii로 촬영한 fits 파일이 있습니다.<br/>
