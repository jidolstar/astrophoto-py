# astrophoto-py
천체사진을 Python으로 보정해 보는 코드를 ChatGPT 4.0과 함께 짜봤습니다.<br/>
원본 이미지는 resources 폴더에 H-alpha, Oiii, Sii로 촬영한 fits 파일이 있습니다.<br/>
Python 3.9 환경에서 app.py을 실행하면 됩니다. 물론 의존되는 모듈은 다운로드 받아야 하고요.<br/>
